# PacMan

## How To Play

Clone repository
```
git clone https://github.com/richardw55555/PacMan.git
```
Grant Permissions
```
chmod +x PacMan/setup.sh
chmod +x PacMan/main.sh
```
Play
```
PacMan/main.sh
```

## Controls
WASD or arrow keys to move
Escape key to quit
R to restart the level
Shift+R to restart the game
